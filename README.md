# Library Manager

Projet Maven pour gérer une bibliothèque.

## Description
Cette application permet de gérer une collection de livres.

## Fonctionnalités
- Ajouter des livres
- Afficher tous les livres
- Compter les livres
- Rechercher des livres par auteur

## Technologies utilisées
- Java
- Maven
- JUnit
